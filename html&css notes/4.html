• MARGIN PADDING BORDER

    1. adjusting

      padding (relleno): space between the element and its border

      margin (margen): distance between it's border and surrounding elements, it can be negative

    2. clockwise notation

      can specify padding / margin (top, right, bottom, left) all in one line:

      .class-name{ padding: 10px, 20px, 10px, 20px; margin: 10px, 20px, 10px, 20px; }

    3. Style html body element (inherit styles)

      Every HTML page has a body element

      body { background-color: black; }
